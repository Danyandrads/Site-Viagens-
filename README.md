✈️ Site de Viagens Responsivo
Site multi-páginas desenvolvido com HTML5, CSS3 e Bootstrap 5, com foco em design responsivo, identidade visual consistente e boa experiência de navegação. Avaliado como o melhor design da turma pelo professor.
---
📄 Páginas
Página	Descrição
Home	Apresentação principal com hero, localização e redes sociais
Destinos	Galeria de destinos com imagens e descrições
Sobre Mim	Apresentação pessoal da criadora do site
Contato	Formulário de contato
---
🛠️ Tecnologias utilizadas
HTML5 — estrutura semântica das páginas
CSS3 — estilização personalizada e responsividade
Bootstrap 5 — grid e componentes de layout
JavaScript — interações e menu responsivo
---
✨ Destaques
Design responsivo para desktop, tablet e mobile
CSS próprio integrado ao Bootstrap para identidade visual única
Integração com Google Maps e redes sociais
Menu de navegação com colapso em telas menores
